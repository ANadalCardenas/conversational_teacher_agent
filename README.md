# Prerequisites

* Ubuntu 24.04 host machine
* Docker
* NVIDIA Container Runtime


# Installation

1. Clone this repository in your machine

```bash
git clone ...
```

2. Build the docker images

```bash
cd conversational_teacher_agent
docker compose build
```

# Launching

1. Launch the docker containers (front-end & back-end):

```bash
cd conversational_teacher_agent
docker compose up
```

2. In your host machine, open a web browser and go to `http://localhost`
